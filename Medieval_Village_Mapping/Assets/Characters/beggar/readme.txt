This pack contains a medieval character picturing a poor guy. Animations are made to let him be a farmer, a tavern barkeeper, a smith, a beggar or a priest. There are 2 different heads and different tools like a hammer or a a brush. There is an optional cloak and he can have hairs or be bold to create even more variations.

Animations are splitted. So it is possible to start each sequence separated like:
animation.Play("Walk");
